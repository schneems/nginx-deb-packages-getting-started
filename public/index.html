Getting started with Heroku .deb Packages: Nginx

This is a sample Nginx application deployed to Heroku. It is served built using the Ubuntu Nginx Debian package built via Heroku .deb Packages Cloud Native Buildpack

This buildpack is available for use on Heroku's Fir generation

How this sample app works

  • This app was deployed to Heroku using Git.
  • When Heroku received the source code, it fetched all the dependencies in the project.toml with the [com.heroku.buildpacks.deb-packages] key , creating a deployable build artifact.
  • The platform then spins up a dyno, a lightweight container that provides an isolated environment in which the build artifact can be mounted and executed.
  • You can scale your app, manage it, and deploy over 150 add-on services, from the Dashboard or CLI.
